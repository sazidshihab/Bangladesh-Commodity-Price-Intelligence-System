1. Here, all folders contain a pickle file generated from python script.
2. The D=0 and D=1 folder contains fitted model pickle files. This means all those pickle files are models themselves after training. This folder also contains SS of each model summary.
3. TSA folder contains a Time Series Decomposition model for each product. It's to see the basic Trend, Seasonalities and Resid for each product.
4. ACF and PACF folder contains ACF and PACF plots in a pickle format of all products.
5. All pickle files are ready to use as a model or plot.
